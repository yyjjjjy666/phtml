# Visitor IP Footer

## Requirements

All requirements for this feature have been removed. The visitor IP footer no longer exists.

The footer has been replaced by the dedicated `/tools/ip` page which shows full IP and geolocation information. Visit `geller.ee/tools/ip` to see IP and geolocation information.

`js/visitor-info.js` is deleted; use `js/ip-tool.js` on the ip tool page only.
